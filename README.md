# Responsive Weather App UI

`[HTML5]` `[CSS3]` `[Flexbox]` `[Responsive Design]` `[SVG]`

This project is a high-fidelity, visually appealing **UI-only** prototype for a weather application. It was built from scratch using only HTML and CSS, focusing on modern styling techniques, responsive layout, and a clean user experience.

This is a strong portfolio piece for demonstrating front-end styling and layout skills, as it shows an understanding of color, spacing, typography, and responsive design.

## ‚ú® Features

* **Modern Gradient UI:** Uses a `linear-gradient` background to create a visually engaging card.
* **Flexbox Layout:** The search bar and weather details sections are built with **CSS Flexbox** for perfect alignment and flexible spacing.
* **Inline SVG Icons:** Uses embedded SVGs for all icons (search, weather, details). This is a modern, high-performance alternative to icon fonts, as it requires no external libraries and the icons are infinitely scalable and stylable with CSS.
* **Fully Responsive:** The layout fluidly adapts to all screen sizes. On narrow screens, the "Weather Details" section neatly stacks vertically.
* **Glass-like Details:** The details section uses an `rgba` background to create a semi-transparent "frosted glass" effect.
